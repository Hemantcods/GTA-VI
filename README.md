# GTA VI landing page with animations

This GTA VI fontend is made with the youtube [tutorial](https://www.youtube.com/watch?v=tccIAsqFgG0&list=PLPx5gMoEi00RyDW0TwD6uU0JOk0QT78wx) 

Link to official repo from youtube [video](https://github.com/asynchronousJavascriptor/GTAVI)

This projet includes the use of following packages:

- GSAP

  - GSAP/React

- Tailwind CSS

## Starting the project:

```
git clone https://github.com/Hemantcods/GTA-VI.git
```
